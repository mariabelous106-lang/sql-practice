# SQL Practice for QA

SQL-запросы для тестировщика: JOIN, GROUP BY, подзапросы, UNION, транзакции.

## Файлы

- `queries.py` — базовые запросы (JOIN, GROUP BY, HAVING, LEFT JOIN)
- `advanced_queries.py` — продвинутые (подзапросы, UNION, транзакции с ROLLBACK)

## Запуск

```bash
python queries.py
python advanced_queries.py
